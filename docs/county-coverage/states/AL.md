## Status Rules

- `verified-present`: generated from imported county presence sources or manual authoritative presence overrides.
- `verified-absent`: only for a county and species when a reputable source explicitly supports absence.
- `not-detected`: only for a county and species when a reputable survey or monitoring source looked and did not detect it.
- `unknown`: the default for every county-species pair that has no verified status yet.

Unknown does not mean absent. It means no defensible county-species determination has been added yet.
