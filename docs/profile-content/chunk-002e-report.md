## Contradictions Resolved

Several registry-facing names were too awkward or too narrow to leave unchanged in public-facing curated pages.

- `Ranunculus ficaria` entered the queue as `Fig buttercup`, but the curated profile uses `Lesser Celandine`.
- `Neonectria faginata` entered the queue as `Beech bark canker`, but the curated profile uses `Beech Bark Disease` because the public-facing issue is the broader disease complex.
- `Iris pseudacorus` entered the queue as `Yellow flag`, but the curated profile uses `Yellow Flag Iris` so the name reads clearly without extra context.

Those are deliberate clarity fixes, not accidental rewrites.

## What Improved

- The project reached the planned `100` curated-profile milestone without dropping back to generic filler.
- The batch balanced woody invaders, aquatic plants, crop and garden diseases, forest pests, and public-facing wildlife species instead of padding the count with one category.
- This pass handled another round of public-facing naming cleanup while also correcting some awkward raw grouping expectations through curated display choices.
- Action guidance stayed conservative where it needed to, especially for aquatic spreaders, livestock-toxic plants, forest pests, and crop diseases.

## What Worked Well

- University of Minnesota Extension remained the strongest single source family for plant and disease profiles.
- Michigan invasive-species pages were especially useful for aquatic plants, Japanese knotweed, and lesser celandine because they combine public ID cues with report-first guidance.
- State and federal forestry sources gave enough detail to keep forest pest and disease profiles specific instead of vague.
- Wildlife profiles stayed readable when the source had a clear public action path, such as boat sanitation, non-transport, or range-edge reporting.

## Weak Spots In This Batch

- Some of the tree and forest-pest profiles still depend on narrower regional sources because national plain-language guidance is uneven.
- Agricultural disease profiles are inherently more specialized than yard or trail species, so their public action guidance has to stay narrower and more diagnostic.
- County coverage remains uneven for a few high-quality profile candidates such as Japanese knotweed and lesser celandine, which means source readiness mattered more than mapped prevalence.

## Process Lessons

- Hitting a round-number milestone only worked because source readiness was checked first. If that discipline slips, the next tranche will drift fast.
- Public-facing common-name cleanup is still worth doing whenever the raw registry name would make an ordinary reader pause or misread the threat.
- Mixed batches stay healthier when each non-plant species still gives the public a concrete action path rather than just an interesting biology lesson.
- State invasive-program pages are especially valuable when they combine look-alike notes, reporting channels, and spread-prevention language on one page.

## Recommendation For The Next Batch

The `100`-profile tranche is complete. The next move should not be a victory lap.

Recommended next step:

1. Start the next tranche with another source-ready batch in the `18 to 25` range instead of assuming every remaining candidate is ready now.
2. Keep prioritizing species where public-facing naming or grouping cleanup will materially improve usability.
3. Continue giving aquatic hitchhikers, tree pests, and diseases conservative action language rather than pretending all readers can remove them directly.
4. Preserve the ledger and report loop exactly as-is because it is now doing real quality-control work, not just documentation.
