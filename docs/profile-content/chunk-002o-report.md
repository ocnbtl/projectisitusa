# Chunk 2O Report

## Scope

Chunk `2O` is a wildlife-heavy category-repair pass built around a denser official source ecosystem rather than a forced mixed backlog. It adds `8` curated profiles and raises the curated total from `302` to `310`.

Profiles added in this batch:

- European Spruce Sawfly
- Rhesus Macaque
- New Guinea Flatworm
- Oscar
- Spotted Tilapia
- Black Acara
- Suckermouth Catfish
- Channeled Applesnail

## Contradictions Resolved

The main contradiction after `2N` was whether the project had enough non-plant source depth to move immediately into another serious chunk without falling back into weak, repetitive backups.

This batch resolves that by narrowing the promise:

- it does not pretend the entire mixed backlog is suddenly deep
- it does prove there is a usable wildlife-led source ecosystem around Florida and southeastern surveillance species
- it keeps the shortlist honest by excluding thinner reptile and disease candidates that still do not have equally strong public-facing guidance

There was also a second contradiction around the kind of species that deserve curation next:

- some remaining candidates are nationally mapped but still weak in public guidance
- others are more geographically concentrated but much better documented by state fish and wildlife agencies, extension, parks, or federal surveillance sources
- this chunk chose the stronger writing ecosystem over the illusion of broader distribution

## What Improved

- Wildlife coverage gained six more curated profiles that are not just mammals. The catalog now has stronger fish, mollusk, primate, and flatworm representation instead of letting wildlife collapse into a feral-swine-heavy pattern.
- Official Florida wildlife sources proved they can support a real cluster. `Oscar`, `Spotted Tilapia`, `Black Acara`, and `Suckermouth Catfish` all benefit from the same FWC plus USGS NAS stack while still reading as distinct profiles.
- The project added a more public-safety-oriented wildlife profile with `Rhesus Macaque`, where conservative language matters more than broad county totals.
- `European Spruce Sawfly` and `New Guinea Flatworm` keep the batch from becoming only warm-water fish and show that smaller non-plant clusters are still possible when the source path is clear.

## Weak Spots In This Batch

- `2O` is still geographically uneven. Much of the strongest action language in this batch comes from Florida or southeastern sources, which means the chunk is category-repair heavy but not nationally balanced.
- `European Spruce Sawfly` has solid government forestry sourcing, but the public-facing action path is necessarily lighter than the strongest homeowner-oriented extension profiles.
- `New Guinea Flatworm` is important and sourceable, but the practical action language is still weighted toward reporting and movement prevention rather than direct household treatment.

## Process Lessons

- The next path to `20 to 30` is not a generic mixed shortlist. It is a larger, clearly bounded source ecosystem with repeated high-quality official guidance, just as the recent plant clusters relied on Virginia Tech Weed ID and this batch relies heavily on FWC plus USGS NAS.
- Wildlife can scale if the shortlist is assembled around a real agency stack instead of around whichever species happen to have photos.
- County totals alone are still not enough. The best chunk candidates remain the ones that also have credible action framing, regulated movement language, or strong public identification guidance.

## Recommendation For The Next Batch

1. Treat `2O` as proof that another wildlife-led pass is possible, but only if it stays tied to strong official source ecosystems.
2. Do not promise a `20 to 30` mixed-category batch yet. Promise a larger cluster only after the shortlist is actually source-vetted.
3. Run the county-audit track in parallel, because better state and county reporting sources may improve both future profile selection and the accuracy of county-level wildlife display.
