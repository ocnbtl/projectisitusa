## Contradictions Resolved

Several raw registry names were too awkward, too narrow, or too technical for public-facing profile pages.

- `Lonicera x bella` entered the batch as `Showy fly honeysuckle`, but the curated profile uses `Bell's Honeysuckle`.
- `Megacopta cribraria` entered the batch as `Bean plataspid`, but the curated profile uses `Kudzu Bug`.
- `Myriophyllum aquaticum` entered the batch as `Parrot feather watermilfoil`, but the curated profile uses `Parrot Feather`.
- `Eichhornia crassipes` entered the batch as `Common water hyacinth`, but the curated profile uses `Water Hyacinth`.
- `Nymphoides peltata` entered the batch as `Yellow floatingheart`, but the curated profile uses `Yellow Floating Heart`.
- `Albizia julibrissin` entered the batch as `Silktree`, but the curated profile uses `Mimosa Tree`.

Those are deliberate clarity fixes, not accidental rewrites.

## What Improved

- The project moved past the `100`-profile milestone without slipping back into filler species selected only for county count.
- This batch rebalanced the workflow toward a stronger aquatic mix while still adding woody invaders, an ornamental tree, a legume pest, an aphid, and another invasive carp profile.
- The naming cleanup work is still paying off because several registry common names would have reduced plain-language usability.
- Report-first guidance stayed strict for aquatic species and crop pests, which kept the public action advice grounded.

## What Worked Well

- Michigan invasive-species pages remain a strong backbone for aquatic plants because they combine look-alikes, impact framing, and reporting paths.
- Tennessee and Minnesota continue to work well together for woody invaders and hedgerow species.
- The workflow still scales when a grouped source supports multiple distinct species, as long as the final copy stays species-specific.
- The post-`100` tranche did not need a reset. The same ledger, checker, and report loop still holds up.

## Weak Spots In This Batch

- The batch leans hard on aquatic plants because that is where high-signal official public guidance remains easiest to use.
- A few species in this pass depend on narrower or more regional sources than ideal, especially when the best public-facing explanation sits in one state or extension system.
- White poplar and common periwinkle are useful public-facing profiles, but the source ecosystem around them is weaker than for the best aquatic and regulatory species.

## Process Lessons

- After `100`, the best protection against drift is still species selection discipline, not faster drafting.
- Grouped official pages can support several related species, but they should only be used when each profile still gets a distinct identification and impact section.
- Aquatic watch-list and prohibited species continue to justify curation even with lower county counts because the public prevention path is clear and important.
- Naming cleanup should keep happening whenever the registry label sounds like database language instead of something an ordinary reader would recognize.

## Recommendation For The Next Batch

The next sub-batch can stay in the `18 to 25` range if the same source quality holds.

Recommended next step:

1. Keep mixing strong aquatic profiles with select woody and wildlife species so the tranche does not collapse into a one-category run.
2. Keep using curated public-facing names when the raw registry wording would confuse non-specialists.
3. Continue treating crop pests, aquatic hitchhikers, and invasive carp with conservative, non-casual action language.
4. Watch for a new weak point after this batch: grouped-source overuse. Reuse is fine, but only if the final copy still reads as species-specific and not mass-produced.
