## Contradiction Resolved

The original working shortlist included common teasel, but the actual source-ready species present in the dataset with image support was cutleaf teasel.

That was not smoothed over. The batch uses cutleaf teasel and the source ledger now reflects that decision explicitly.

## What Improved

- This is the first follow-up batch that moved back into the target `18 to 25` range.
- The batch balances invasive plants with a disease profile, an indoor nuisance insect, and a public-health mosquito.
- Several profiles now use public-facing common names that are clearer for everyday readers than the raw registry wording.
- The source ledger now covers more management patterns, including container-source mosquito control and indoor exclusion guidance.

## What Worked Well

- University of Minnesota Extension remained the strongest anchor for invasive plant profiles.
- Penn State and Oklahoma State filled important gaps for Japanese stiltgrass, jimsonweed, and sericea lespedeza.
- CDC worked well for mosquito guidance because the public action path is strongly prevention-focused and does not require local speculation.
- The workflow handled the larger batch without breaking the profile checker.

## Weak Spots In This Batch

- The batch is still plant-heavy because that is where source-ready, contributor-friendly guidance is easiest to verify and adapt.
- A few species in the underlying registry use less familiar common names, so the curated copy now carries the clearer public-facing names instead.
- Butternut canker guidance is intentionally conservative because documentation and conservation value matter more than improvised treatment.

## Process Lessons

- Scaling up works when the species list is selected for source readiness first, not just county coverage.
- Public-facing common names sometimes need to be curated manually to keep the site understandable.
- The source ledger is now doing real project work by preserving why a species made it into a batch and which institutions supported the wording.
- Mixed batches improve the workflow more than plant-only batches, but only if the non-plant species still have strong public guidance sources.

## Recommendation For The Next Batch

The next sub-batch can stay in the `18 to 25` range.

Recommended next step:

1. Keep at least a small cross-category mix.
2. Prioritize species whose registry common names are confusing and need clearer public-facing treatment.
3. Keep using conservative action language for toxic species, disease profiles, and public-health vectors.
4. Continue source logging and reporting in the same drafting pass so scale does not outrun provenance.
