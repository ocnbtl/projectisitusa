## Contradictions Resolved

The raw registry names were not left untouched where they were poor public-facing choices.

- `Persicaria perfoliata` entered the batch as `Asiatic tearthumb`, but the curated profile uses `Mile-a-Minute Vine`.
- `Salvinia molesta` entered the batch as `Kariba weed`, but the curated profile uses `Giant Salvinia`.
- `Amynthas agrestis` entered the batch as `Jumping snake worm`, but the curated profile uses `Jumping Worm`.

Those are deliberate clarity fixes, not accidental rewrites.

## What Improved

- This batch kept the `18`-profile scale that worked in Chunk 2C without dropping back to a tiny pilot.
- The plant-heavy core is still there, but the batch now also covers a termite, an ornamental moth, a forest disease, a reef fish, a freshwater snail, and a soil-altering earthworm.
- The workflow handled more public-facing naming cleanup without blurring the underlying scientific records.
- Aquatic prevention language is stronger in this batch because several profiles depend on transport sanitation more than direct removal.

## What Worked Well

- Penn State remained a strong anchor for woody vine and shrub profiles.
- Tennessee and California state resources helped with species that are common in public guidance but not always covered cleanly in a single extension page.
- NOAA and NPS were effective for wildlife profiles where the public action path is mostly prevention, gear sanitation, or legal removal.
- The checker still passed with the larger content block and the no-em-dash rule intact.

## Weak Spots In This Batch

- The batch still leans toward plants because that is where source-ready public guidance remains deepest.
- Several woody species share a similar control pattern, so future chunks should keep watching for copy drift even when the species themselves are different.
- Some of the non-plant species are regionally concentrated, which is acceptable for now because they still have clear public action paths and image support.

## Process Lessons

- Public-facing common names need active curation whenever the registry wording would confuse ordinary readers.
- Mixed-category batches stay useful when every non-plant profile still has a concrete public action path rather than just interesting biology.
- Aquatic and soil-spread species benefit from very explicit sanitation language because that is the main practical action many readers can actually take.
- It was worth running an early checker pass before updating docs because it confirmed the large content patch was structurally sound.

## Recommendation For The Next Batch

The next sub-batch can stay in the `18 to 25` range if source readiness holds.

Recommended next step:

1. Keep prioritizing species that need public-facing naming cleanup.
2. Add at least one more disease or forest-pest profile if the sources are good enough.
3. Continue treating aquatic hitchhiker species and risky pathogens with conservative action language.
4. Keep the ledger and report updates in the same pass so provenance stays coupled to drafting.
