# Chunk 2B Report

## Scope

Chunk 2B added `12` more curated profiles inside the broader `100`-profile tranche.

Profiles added or upgraded in this batch:

- White-pine blister rust
- Yellow nutsedge
- Autumn olive
- White mulberry
- Glossy buckthorn
- Cogongrass
- Saltcedar
- European starling
- Asiatic clam
- Wild parsnip
- Poison hemlock
- Spotted knapweed

## Contradiction Resolved

The older Chunk 2 plan still described the `100` tranche as four fixed `25`-profile sub-batches.

That no longer matched the actual workflow. Instead of smoothing that over, the plan was corrected to a variable-size ramp-up model. Smaller batches are allowed while the workflow is still improving, and later batches can scale back up once validation and source quality remain stable.

## What Improved

- The source mix is stronger and more varied than in Chunk 2A.
- More profiles now pair a practical extension source with a broader federal or program source.
- The action guidance is more conservative for toxic plants, riparian invaders, and infrastructure-linked aquatic species.
- The source inventory now captures a wider range of profile use cases, including woody control, toxic-plant safety, and aquatic spread prevention.

## What Worked Well

- University extension pages remained the best source of plain-language public action guidance.
- Federal and program-level pages worked well for origin, spread pathways, and why-the-species-matters framing.
- A mixed batch across plants, disease, birds, and aquatic invertebrates improved the workflow more than another plant-heavy run would have.
- The no-em-dash rule remained easy to enforce with the profile checker in place.

## Weak Spots In This Batch

- Yellow nutsedge still has a more complicated native-versus-introduced history than most species in the tranche, so the origin framing had to stay cautious.
- Saltcedar management remains site-sensitive, especially in riparian systems where habitat tradeoffs matter.
- European starling action guidance is useful, but it is less report-driven than many of the plant and pest profiles, so the contact path is more technical-support oriented than detection oriented.

## Process Lessons

- Pairing one practical source with one contextual source is a strong default for many species.
- Toxic and phototoxic plant profiles need extra discipline so the `What to do` section stays helpful without turning reckless.
- A batch does not need to be large to improve the workflow. It needs to stress a new part of the workflow.
- The source ledger is now valuable enough that it should be treated as required project infrastructure, not optional documentation.

## Recommendation For The Next Batch

The next sub-batch should move upward, but not jump straight to a huge pass.

Recommended next step:

1. Target `18 to 25` profiles.
2. Favor species that already have at least one strong extension or agency page.
3. Keep the category mix broad so the workflow does not overfit to terrestrial plant profiles.
4. Continue updating the source ledger and report in the same pass as the drafting.
