## Contradictions Resolved

The active plan after `2K` still favored mixed category repair, especially for non-plant gaps. The user then asked for `30` profiles in one pass.

That created a real contradiction:

- the source-safe backlog was not balanced across plants, insects, wildlife, and fungi-diseases
- the strongest ready ecosystem for a same-pass `30` was Penn State Plant Identification
- forcing category balance would have pushed the batch toward thinner sources and flatter copy

This chunk resolves that directly instead of smoothing it over:

- `2L` is intentionally plant-heavy
- the batch is grouped around one verified source ecosystem that can sustain species-specific copy at this size
- the category-balance goal is deferred, not denied

There was also a public-name cleanup pass inside this chunk:

- `Persicaria maculosa` now uses `Lady's Thumb` instead of `Spotted ladysthumb`
- `Artemisia vulgaris` now uses `Mugwort` instead of `Common wormwood`
- `Rumex obtusifolius` now uses `Broadleaf Dock` instead of `Bitter dock`
- `Barbarea vulgaris` now uses `Yellow Rocket` instead of `Garden yellowrocket`
- `Lotus corniculatus` now uses `Bird's-foot Trefoil` instead of `Garden bird's-foot-trefoil`
- `Senecio vulgaris` now uses `Common Groundsel` instead of `Old-man-in-the-spring`

## What Improved

- The catalog gained a full block of practical, image-backed common weed profiles that were still missing despite strong county coverage.
- The grouped-source strategy proved it can scale past the earlier `18 to 25` comfort range when the source ecosystem is consistent and the species are chosen carefully.
- Several weak public labels were corrected in the curated layer, which improves browsing and page credibility without changing the underlying registry identity.
- The new batch fills obvious lawn, pasture, roadside, nursery, and disturbed-ground gaps that users are likely to encounter in daily life.

## Weak Spots In This Batch

- This pass is more plant-heavy than the plan ideally wanted, and that imbalance is a real tradeoff rather than a hidden one.
- Because the batch leans on one source ecosystem, future chunks should not assume the same scale is safe for insects, wildlife, or fungi-diseases.
- `Red Fescue` is the softest fit in the group because its invasive framing depends more on site goals and seeded-cover persistence than on obvious high-impact spread.

## Process Lessons

- A `30`-profile pass is quality-safe when it is built on a strong clustered source ecosystem, not when it tries to force even category coverage.
- The grouped-source approach still needs species-specific editing discipline or the copy will flatten quickly at this size.
- Common-name cleanup remains one of the highest-value side effects of curation because several registry labels are still poor public UX.

## Recommendation For The Next Batch

1. Do not treat `30` as the new default batch size for every chunk.
2. Return to a smaller mixed batch next, with category repair taking priority once the source stack is strong enough again.
3. Keep using clustered source ecosystems when they are real, but make the category tradeoff explicit every time.
