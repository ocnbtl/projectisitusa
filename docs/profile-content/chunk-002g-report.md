## Contradictions Resolved

Several raw registry names still read like database labels, spacing artifacts, or incomplete public names. This batch fixes them directly instead of carrying them forward.

- `Hypericum perforatum` entered the batch as `Common St. Johnswort`, but the curated profile uses `Common St. John's Wort`.
- `Melia azedarach` entered the batch as `Chinaberrytree`, but the curated profile uses `Chinaberry Tree`.
- `Ricinus communis` entered the batch as `Castorbean`, but the curated profile uses `Castor Bean`.
- `Impatiens glandulifera` entered the batch as `Ornamental jewelweed`, but the curated profile uses `Himalayan Balsam`.
- `Salvator merianae` entered the batch as `Black and white tegu`, but the curated profile uses `Argentine Black and White Tegu`.
- `Sirex noctilio` entered the batch as `Woodwasp`, but the curated profile uses `Sirex Woodwasp`.
- `Pyrrhalta viburni` entered the batch as `Vibernum leaf beetle`, but the curated profile uses `Viburnum Leaf Beetle`.

Those are deliberate public-facing corrections, not accidental rewrites.

## What Improved

- This is the first batch in the current workflow to scale cleanly to `24` profiles without dropping back into filler-heavy species selection.
- The source strategy improved. Instead of only clustering by taxonomic similarity, this pass grouped species by source ecosystem that already had strong public guidance, which made it possible to move faster without writing flatter copy.
- Category balance held up even with a larger batch. The tranche includes woody plants, wetland and aquatic plants, crop pests, tree pests, a reptile, a fish, and a mollusk.
- The naming-cleanup work kept paying off. Several registry labels were still too awkward for public pages and would have lowered confidence if left untouched.

## What Worked Well

- Pennsylvania, Minnesota, and Wisconsin extension sources remain strong for insects and plant profiles because they usually include identification, timing, and practical action framing in one place.
- Florida and federal invasive-species pages work well for high-visibility southern invaders where public reporting is part of the response.
- A larger batch is realistic when the species are selected around strong source clusters first and county count second.
- Public-facing action guidance stayed conservative even with the bigger scope. Toxic plants, invasive reptiles, fish, and tree pests all kept a report-first or timing-sensitive tone.

## Weak Spots In This Batch

- A few profiles rely on one main official source rather than a stronger two-source pairing. The content is still supportable, but the source base is thinner than the best batches.
- The batch includes more ornamental escapees than 2F did, which is useful for homeowners but can make the tranche feel less regulatory than the aquatic watch-list passes.
- The scaled-up size increases the risk of sentence-pattern repetition. It did not break quality here, but it is now a real watch item.

## Process Lessons

- The best way to scale is not to lower the bar. It is to choose species that share strong source ecosystems while still needing clearly different identification and impact copy.
- Larger batches work better when they mix two or three strong clusters rather than trying to balance every category evenly from the start.
- Registry name cleanup is still not optional. Scaling up exposes naming defects faster because the weak labels pile up.
- A `24`-profile pass is workable, but only if the final validation loop is still treated as mandatory and not as a paperwork step.

## Recommendation For The Next Batch

The next sub-batch can stay in the `22 to 28` range if source quality remains this strong.

Recommended next step:

1. Keep using grouped official sources as a speed lever, but only when the resulting profiles still read as distinct species pages.
2. Favor species with clear homeowner, landowner, angler, or field-scout relevance over broad low-signal weeds that only look attractive by county count.
3. Keep correcting raw registry names whenever they are visibly unfit for public-facing pages.
4. Watch the next scaling risk closely: not source scarcity this time, but repetition in action language and sentence rhythm across a larger tranche.
